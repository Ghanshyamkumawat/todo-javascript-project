# To-Do JavaScript Project

A simple To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Saves data using Local Storage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open `index.html` in any browser
